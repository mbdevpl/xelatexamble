% adjust look and feel of slide header and footer, which might or might not contain the:
%  - section/subsection info
%  - presentation title
%  - author
%  - page number
% one of: infolines, miniframes, shadow, sidebar, smoothbars, smoothtree, split, tree

% remove navigation symbols
